# Cronograma del Proyecto
  
El proyecto se ejecuta en **5 Sprints** distribuidos en 3 hitos de evaluación.  
  
## Resumen de hitos
  
| Hito | Fecha | Peso | Sprints cubiertos |
|---|---|---|---|
| Hito 1 | 13.MAY.2026 | 15% (65% sobre EC1) | Sprint 0 |
| Hito 2 | 10.JUN.2026 | 60% (85% sobre EC2) | Sprints 1 y 2 |
| Hito 3 | 13.JUL.2026 | 100% (100% sobre EC3) | Sprints 3 y 4 |
  
## Detalle por Sprint
  
### Sprint 0 — Planificación  
**Período:** hasta 16.MAY.2026  
**Líder:** Jerry Anderson Huaynacho Mango  

**Objetivos:**
- Seleccionar producto open source (AppFlowy-Cloud)  
- Crear Organization, fork y Project en GitHub  
- Activar GitHub Pages con documentación inicial  
- Definir Product Backlog inicial (historias de usuario)  
- Asignar roles y rotación de liderazgo  
  
### Sprint 1 — Análisis y configuración DevOps inicial  
**Período:** 17.MAY.2026 – 27.MAY.2026  
**Líder:** Giovani Angel Mendoza Contreras  

**Objetivos:**
- Análisis arquitectónico de AppFlowy-Cloud  
- Levantar el entorno local con Docker Compose  
- Configurar primer workflow en GitHub Actions (build + lint)  
- Definir alcance de mejoras / refactorización  
  
### Sprint 2 — CI/CD funcionando (Hito 2)  
**Período:** 28.MAY.2026 – 10.JUN.2026  
**Líder:** José María Ticona Saure  

**Objetivos:**

- Despliegue automatizado a entorno de staging  
- Documentación técnica en GitHub Pages

- Entrega del Hito 2: producto funcionando con CI/CD  
  
### Sprint 3 — Mejoras y refactorización  
**Período:** 11.JUN.2026 – 26.JUN.2026  
**Líder:** Gabriel Quispe Mamani  

**Objetivos:**

- Mejorar pipeline de despliegue  
  
### Sprint 4 — Cierre y artículo IEEE  
**Período:** 27.JUN.2026 – 13.JUL.2026  
**Líder:** Jerry Anderson Huaynacho Mango  

**Objetivos:**
- Despliegue CI/CD completo y automatizado  
- Documentación técnica final del proceso ágil Scrum + DevOps

- Presentación final (Hito 3)  
  
## Ceremonias Scrum por Sprint
  
| Ceremonia | Frecuencia | Responsable |
|---|---|---|
| Sprint Planning | Inicio de cada Sprint | Líder del Sprint |
| Daily Stand-up | Diario (asíncrono por chat) | Todo el equipo |
| Sprint Review | Final de cada Sprint | Líder del Sprint |
| Sprint Retrospective | Final de cada Sprint | Todo el equipo |

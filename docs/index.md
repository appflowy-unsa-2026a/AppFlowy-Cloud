## Producto de software seleccionado

**AppFlowy-Cloud** — backend self-hosted del workspace colaborativo open source [AppFlowy](https://appflowy.io), alternativa a Notion.

| Aspecto | Detalle |
|:---|:---|
| Repositorio original | [AppFlowy-IO/AppFlowy-Cloud](https://github.com/AppFlowy-IO/AppFlowy-Cloud) |
| Fork del equipo | [appflowy-unsa-2026a/AppFlowy-Cloud](https://github.com/appflowy-unsa-2026a/AppFlowy-Cloud) |
| Licencia | AGPL-3.0 _(excepción aprobada por el docente)_ |
| Stack tecnológico | Rust · TypeScript · Docker Compose |
| Último release | v0.11.9 — 12 de mayo de 2026 |

{: .note }
> El producto fue seleccionado tras verificar que cumple los criterios de la ficha: licencia open source, dominio empresarial, stack moderno, complejidad superior a 10 KLOC, soporte de Docker Compose y actividad reciente sostenida.
